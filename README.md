## get it up and running.

``` bash
# go into the project
cd vue-laravel-image-uploader

# create a .env file
cp .env.example .env

# install composer dependencies
composer update

# install npm dependencies
npm install

# generate a key for your application
php artisan key:generate

# run webpack and watch for changes
npm run watch

```
